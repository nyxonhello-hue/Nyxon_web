🐧 Getting started on Linux

1
Download the .AppImage
Click the Download button above to get the latest AppImage file.
2
Make it executable
Right-click → Properties → Permissions → tick "Allow executing as program". Or run:
chmod +x Nyxon*.AppImage
3
Double-click to launch
That's it. No installation needed. Nyxon runs directly from the AppImage.
